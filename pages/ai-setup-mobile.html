AI 비서 활성화 완료
OpenRouter API Key 발급 방법
OpenRouter는 카드 등록 없이 DeepSeek, Llama, Qwen 등 여러 무료 AI 모델을 키 하나로 이용할 수 있는 서비스입니다. 아래 버튼을 누르면 새 탭에서 OR이 열립니다. 키를 복사하는 순간 이 화면으로 자동 복귀됩니다.
단계별 안내
1
openrouter.ai 접속
위 버튼을 누르면 새 탭에서 openrouter.ai/keys가 열립니다. 메인 화면 상단의 Get API Key 버튼을 클릭합니다.
2
로그인 방법 선택
GitHub, Google, 이메일 중 하나로 로그인합니다. Google이 가장 빠릅니다.
3
Google 계정 이메일 입력
Google 계정 이메일(또는 전화번호)을 입력하고 다음을 클릭합니다.
4
비밀번호 입력
해당 Google 계정 비밀번호를 입력하고 다음을 클릭합니다.
5
정보 제공 동의
이름·이메일을 확인할 수 있도록 허용하는 화면입니다. Continue를 클릭합니다.
Google 계정 표준 보안 절차입니다. 비밀번호는 OpenRouter로 전달되지 않습니다.
6
약관 동의
Terms of Service, Privacy Policy, Model Terms에 동의 체크 후 Continue를 클릭합니다.
7
API Keys 페이지로 이동
가입 후 openrouter.ai/keys로 이동합니다. 왼쪽 위 + New Key 버튼을 클릭합니다.
8
새 Key 생성
Name에 구분할 이름(예: hondi)을 입력하고 Create를 클릭합니다. 나머지는 기본값으로 두면 됩니다.
9
발급된 Key 복사
sk-or-v1-로 시작하는 키가 한 번만 표시됩니다. 복사 버튼을 눌러 즉시 복사하세요. 복사하는 순간 이 화면으로 자동 복귀됩니다. 이 창을 닫으면 다시 볼 수 없습니다.
10
자동 완료
키가 자동으로 입력되고 저장됩니다. 이 탭이 닫히면 혼디 대화창에서 AI 비서가 바로 시작됩니다.
키는 이 기기의 로컬 저장소에만 저장됩니다. 서버로 전송되지 않습니다.
참고
  • 무료 티어 기준 분당 20 요청 / 일 200 요청까지 사용 가능합니다.
  • 무료 모델은 입력 데이터가 제공사 학습에 사용될 수 있습니다. 민감한 개인정보는 입력하지 마세요.
  • 키를 분실했다면 openrouter.ai/keys에서 새 키를 발급하고 이 화면에서 다시 저장하면 됩니다.
