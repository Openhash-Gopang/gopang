나만의 AI 비서 활성화

OpenRouter 키 하나로 여러 AI 모델을 무료로 순차 이용합니다. 한 모델의 무료 한도가 차면 자동으로 다음 모델로 전환됩니다.

🌐 OpenRouter는 키 하나로 DeepSeek, Llama, Qwen 등 여러 개의 무료 AI 모델을 이용할 수 있는 서비스입니다. 카드 등록이 필요 없습니다.
🔄 등록 시점에 현재 가용한 무료 모델을 실시간으로 확인하여 자동 구성하고, 한도가 차면 다음 모델로 전환되어 AI 비서가 끊기지 않고 계속 동작합니다.

OpenRouter API Key 발급 방법

⚠️ 아래 1번 링크를 누르면 이 화면에서 바로 OpenRouter로 이동합니다. Key를 복사한 뒤, 휴대폰 브라우저의 뒤로가기를 눌러 이 화면으로 돌아오세요.

  1. openrouter.ai 접속 → 오른쪽 상단 Sign Up (카드 불필요)
  2. 로그인 후 오른쪽 상단 아이콘 → Keys 메뉴
  3. Create Key 버튼 클릭 → 이름 입력 후 생성
  4. sk-or-v1-... 로 시작하는 키를 즉시 복사 (다시 표시되지 않음)
  5. 뒤로가기로 이 화면에 돌아와 아래 입력칸에 붙여넣고 저장 버튼을 누르세요

🔒 이 키는 이 기기에만 저장되며, 다른 곳으로 전송되지 않습니다.
