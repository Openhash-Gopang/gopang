나만의 AI 비서 활성화

귀하 한 사람만을 위한 맞춤형으로 진화합니다. 모델을 선택하면 해당 업체의 무료 API Key 발급 페이지가 새 탭에서 열리고, 단계별로 안내해 드립니다.

고팡 휴대폰 웹앱에서 인증된 본인만 AI Key를 등록할 수 있습니다.

💡 여러 LLM을 동시에 등록해두시면, 한 모델의 무료 한도가 차더라도 등록 순서대로 다음 모델로 자동 전환되어 AI 비서가 끊기지 않고 계속 동작합니다. 2~3개 모델을 함께 등록하는 것을 권장합니다.
🔒 입력하신 API Key는 이 화면에서 즉시 암호화되어 전송됩니다. 고팡 서버는 평문 Key를 절대 볼 수 없으며, 휴대폰만 복호화할 수 있습니다.
