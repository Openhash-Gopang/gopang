나만의 AI 비서 활성화

OpenRouter 키 하나로 여러 AI 모델을 무료로 순차 이용합니다. 한 모델의 무료 한도가 차면 자동으로 다음 모델로 전환됩니다.

본인의 고팡 아이디(핸들)를 입력하세요.
예: 96627170 또는 @96627170 — 설정한 Key는 이 핸들의 휴대폰만 복호화할 수 있습니다.

🌐 OpenRouter는 키 하나로 DeepSeek, Llama, Qwen 등 여러 개의 무료 AI 모델을 이용할 수 있는 서비스입니다. 카드 등록이 필요 없습니다.
🔄 전송 시점에 현재 가용한 무료 모델을 실시간으로 확인하여 자동 구성하고, 한도가 차면 다음 모델로 전환되어 AI 비서가 끊기지 않고 계속 동작합니다.
🔒 입력하신 Key는 이 화면에서 즉시 암호화되어 전송됩니다. 고팡 서버는 평문 Key를 절대 볼 수 없습니다.

OpenRouter API Key 발급 방법
📋 스크린샷으로 보는 절차 안내 ↗
openrouter.ai/keys ↗
  1. openrouter.ai 접속 → 오른쪽 상단 Sign Up (카드 불필요)
  2. 로그인 후 오른쪽 상단 아이콘 → Keys 메뉴
  3. Create Key 버튼 클릭 → 이름 입력 후 생성
  4. sk-or-v1-... 로 시작하는 키를 즉시 복사 (다시 표시되지 않음)
  5. 아래 입력칸에 붙여넣고 전송 버튼을 누르세요
※ 무료 티어: 분당 20 요청 / 일 200 요청
※ 자동 순환 모델: 전송 시점에 OpenRouter에서 현재 실제로 무료인 모델을 실시간으로 확인하여 자동 구성합니다
※ 무료 모델은 입력 데이터가 제공사 학습에 사용될 수 있으므로 민감한 개인정보는 입력하지 마세요.
