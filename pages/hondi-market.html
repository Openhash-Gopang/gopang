"K" 접두사가 붙은 앱들은 혼디가 제시하는 AI Webapp의 시범 사례이며, hondi.net 위에서 운영되고 있습니다. 사용자가 새로 등록하는 Webapp은 hondi.net 서브도메인을 받게 됩니다. 가격은 모두 1회 이용당 부과되며, 1T(GDC)는 KRW 1,000원에 해당합니다.

등록 절차
나만의 Webapp, 다섯 단계로 게시하기
1

주소 정하기

원하는 이름을 정하면 *****.hondi.net 형태의 고유 주소가 됩니다. 아래 신청서의 주소 입력란에 바로 입력할 수 있습니다.

2

GitHub에 코드 업로드

System Prompt를 포함한 Webapp 코드 전체를 본인 GitHub 저장소에 공개(Public)로 업로드합니다. 혼디장의 모든 Webapp은 예외 없이 오픈소스여야 합니다.

3

혼디 보안 심사

혼디 심사 AI가 제출된 저장소를 직접 방문해 System Prompt와 코드에 악의적인 지시나 코드(데이터 무단 전송, 사용자 기만, 인증·결제 우회 등)가 없는지 점검합니다.

심사 System Prompt 전문 보기 ↗
4

Cloudflare 등록

심사를 통과하면 혼디가 *****.hondi.net 주소를 Cloudflare에 등록합니다. 신청자가 직접 DNS를 설정할 필요는 없습니다.

5

게시 및 과금 시작

게시된 Webapp은 코드가 모두 공개된 오픈소스이지만, 신청서에 기재한 가격대로 실제 이용 1회마다 과금됩니다. 코드를 보는 것은 무료지만, 서비스를 쓰는 것은 유료입니다.

오픈소스 + 사용량 기반 과금

혼디장의 모든 Webapp은 코드와 System Prompt가 누구에게나 공개된 오픈소스입니다. 그러나 공개된 코드를 읽는 것과 실제로 그 서비스를 이용하는 것은 별개입니다. 이용자가 사이트에 접속해 AI 비서를 실제로 호출하면, 등록 시 신청자가 정한 가격에 따라 GDC로 이용료가 이용 1회마다 자동 정산됩니다.

이용자의 AI 비서(AC)가 대신 거래합니다

이용자는 혼디장 화면을 직접 클릭하지 않아도 됩니다. "혼디야, K-Law 써줘"라고 말하면 이용자의 나만의 AI 비서(AC)가 해당 Webapp의 AI를 직접 호출해 업무를 처리하고, GDC 결제 내역과 이용 기록을 이용자의 PDV에 자동으로 남깁니다. 이 기록은 Openhash에 해시가 등록되어 위변조가 불가능한 이용 영수증 역할을 합니다.

혼디의 보안 심사 책임

오픈소스라는 이유로 검증되지 않은 채로 게시되지는 않습니다. 혼디는 등록 신청이 들어올 때마다 신청자의 GitHub 저장소를 방문해 System Prompt와 코드를 심사하며, 통과한 Webapp만 hondi.net 주소를 받고 게시됩니다.
