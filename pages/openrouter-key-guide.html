OpenRouter API Key 발급 절차 안내

🌐 OpenRouter는 카드 등록 없이 DeepSeek·Llama·Qwen 등 여러 무료 AI 모델을 키 하나로 이용할 수 있는 서비스입니다.

아래 9단계를 그대로 따라하면 약 2분 안에 나만의 AI 비서 설정에 필요한 API Key를 발급받을 수 있습니다.

openrouter.ai/keys 바로가기 ↗
1
openrouter.ai 접속
브라우저에서 openrouter.ai로 접속합니다. 메인 화면 상단의 보라색 Get API Key 버튼을 클릭합니다. (회원이 10M명 이상, 모델 400개 이상을 지원하는 검증된 서비스입니다.)
2
로그인 방법 선택
로그인 화면이 뜹니다. GitHub·Google·MetaMask 중 하나로 간편 로그인하거나, 이메일 주소로 가입할 수 있습니다. 가장 빠른 방법인 Google 아이콘을 클릭합니다.
3
Google 계정 이메일 입력
평소 사용하는 Google 계정의 이메일(또는 휴대전화 번호)을 입력하고 다음을 클릭합니다.
4
비밀번호 입력
해당 Google 계정의 비밀번호를 입력하고 다음을 클릭합니다.
5
정보 제공 동의
OpenRouter가 이름·프로필 사진·이메일 주소를 확인할 수 있도록 허용하는 화면입니다. 내용을 확인한 뒤 Continue를 클릭합니다.
이 단계는 Google 계정 자체의 표준 보안 절차로, 비밀번호가 OpenRouter에 전달되지 않습니다.
6
약관 동의
OpenRouter의 Terms of Service, Privacy Policy, Model Terms에 동의 체크 후 Continue를 클릭하면 가입이 완료됩니다.
7
API Keys 페이지로 이동
가입이 끝나면 openrouter.ai/keys 페이지로 이동합니다. 아직 발급된 키가 없으므로 "No API keys yet"라고 표시됩니다. 왼쪽 위 보라색 + New Key 버튼을 클릭합니다.
8
새 Key 생성
Name에 구분할 이름(예: hondi)을 입력합니다. Credit limit은 비워두면 무제한, Expiration도 No expiration으로 그대로 두면 됩니다. 입력 후 Create를 클릭합니다.
9
발급된 Key 복사
sk-or-v1-로 시작하는 키가 단 한 번만 표시됩니다. 오른쪽 복사 아이콘을 눌러 즉시 복사하세요. 이 창을 닫으면 다시 볼 수 없습니다.
복사한 키는 혼디의 나만의 AI 비서 활성화 화면 입력칸에 붙여넣으면 됩니다. 키는 입력 즉시 이 기기에서 암호화되어 전송되며, 평문으로 저장되지 않습니다.
참고
  • 무료 티어 기준 분당 20요청 / 일 200요청까지 사용 가능합니다.
  • 무료 모델은 입력 데이터가 제공사 학습에 사용될 수 있으니 민감한 개인정보는 입력하지 마세요.
  • 키를 분실했다면 이 페이지를 다시 따라 새 키를 발급하면 됩니다 (기존 키는 openrouter.ai/keys에서 삭제 가능).
