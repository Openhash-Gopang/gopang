# 위치기반서비스사업 신고 — 절차 실측 검증 (복합 민원 사례)

> 작성일: 2026-07-10
> 배경: "일부 민원은 여러 증명을 요구한다"는 지적에 대한 실제 사례 검증. 이전 세션(요약본)에서 이미 이 절차를 다뤘으나, 이번에 웹 검색으로 재검증하니 **이전 요약이 놓친 중요한 구분점**을 발견했다.

---

## 핵심 발견 — "위치정보사업"과 "위치기반서비스사업"은 서로 다른 인허가다

이전 세션 요약에는 "위치기반사업자 등록"이라는 명칭으로 뭉뚱그려 있었는데, 실제로 검색해보니 **완전히 다른 두 개의 인허가**가 존재한다.

| 구분 | 위치정보사업 | 위치기반서비스사업 |
|---|---|---|
| 성격 | 위치정보를 **직접 수집**하는 사업(예: 지도업체·통신사) | 이미 수집된 위치정보를 **활용해 서비스만 제공**(예: 배달앱) |
| 인허가 방식 | 허가·등록(더 무거움) | **신고**(더 가벼움) |
| 필요서류 | 신청서, 사업계획서, **법인등기부등본**, **주주명부**(설립예정법인) 등 | 사업계획서, 설비확인서류, 보호조치증명서류 |
| 근거 | 위치정보법 | 위치정보법 제9조 |
| 정부24 안내 | "위치 정보 사업자 등록·신고" | "위치기반서비스사업 신고, 변경신고" |

**Hondi에게 중요한 시사점**: 사용자가 "위치기반 서비스 등록하고 싶어요"라고만 말하면, 이게 자기 위치정보를 직접 수집하려는 건지(위치정보사업, 무거움) 아니면 이미 있는 위치정보를 활용만 하려는 건지(위치기반서비스사업, 가벼움)를 먼저 구분해야 한다 — 이전 세션에서는 이 구분 없이 하나로 뭉쳐서 다뤘을 가능성이 있다.

---

## "위치기반서비스사업 신고"의 실제 절차 — 두 갈래

### 갈래 A: 일반 신고 (법 제9조)

**구비서류 4종**:
1. 사업계획서(사업자 현황 및 사업 내용)
2. 사업용 주요설비의 내용 및 설치장소 확인서류(임차 시 임차 증명서류로 대체)
3. 위치정보 보호조치 증명서류(법 제16조 근거)
4. (외국회사인 경우만) 해당 국가의 법인등기사항증명서·영업소등기사항증증명서

### 갈래 B: 소상공인·1인창조기업 특례 신고 (법 제9조의2) — 훨씬 간단

**구비서류 2종만**:
1. **소상공인확인서** 또는 1인창조기업 확인서류
2. 사업 종류·내용 설명자료

**중요**: 사업 개시 후 1개월 이내에는 신고 없이도 가능하지만, 1개월이 지난 후에도 계속하려면 신고해야 한다는 유예 규정이 있다.

### 공통 사항

- **접수처**: 전자민원센터(emsit.go.kr) 온라인 신고
- **소관부처**: 방송미디어통신위원회(2025-10-01 개편 이후 명칭, 이전엔 방송통신위원회)
- **행정정보공동이용 활용**: 담당공무원이 사업자등록증명·법인등기사항증명서 등을 행정정보공동이용으로 직접 확인 가능 — 신청인이 동의하면 **이 서류들을 직접 제출하지 않아도 된다**(민원처리법 제10조, 불필요한 서류 요구 금지 원칙과 연결).

## 이게 왜 "여러 증명을 요구하는 민원"의 좋은 사례인가

갈래 B(소상공인 특례)만 봐도 이미 **다른 민원의 산출물을 입력으로 요구하는 구조**다:

```
[SP-CIVIL-BIZ 계열 또는 별도 시스템: 소상공인확인서 발급]
                    ↓ (출력이 다음 절차의 입력이 됨)
[위치기반서비스사업 소상공인 특례 신고]
                    ↓
출력: 신고확인증
```

## SP-CIVIL 리프 SP와의 매핑

| 필요 서류 | 담당 SP-CIVIL 리프 (있는 경우) |
|---|---|
| 소상공인확인서 | SP-CIVIL-BIZ-ISSUE(중소기업현황정보시스템 발급, 별도 시스템이라 정부24 직접 관할 아닐 수 있음 — 확인 필요) |
| 사업자등록증명(행정정보공동이용 대상) | SP-CIVIL-BIZ-ISSUE |
| 법인등기사항증명서(외국회사·행정정보공동이용 대상) | SP-CIVIL-FAMILY-ISSUE(옛 COURT 흡수분 — 법원 등기 관련) |
| 위치기반서비스사업 신고 자체 | **SP-CIVIL 64개 목록에 없음** — 방송미디어통신위원회 소관의 개별 신고로, 17개 실측 카테고리 중 어디에도 명확히 안 잡힘(★확인 필요: BIZ로 편입해야 할 수도★) |

**정직하게 밝힘**: "위치기반서비스사업 신고" 자체는 지금까지 만든 64개 SP-CIVIL 어디에도 정확히 대응하지 않는다. 이건 애초에 650건 표본에 이 민원이 걸리지 않았기 때문이다(저빈도 민원). BIZ(사업자 지원) 카테고리로 편입하는 게 합리적으로 보이지만, 확정하려면 추가 확인이 필요하다.

## 정직하게 남긴 것

1. 이전 세션 요약에서 "위치기반사업자 등록"이라 부르던 게 실제로는 두 가지(위치정보사업/위치기반서비스사업) 중 어느 쪽이었는지 이번에 명확히 재검증했다 — 소상공인 특례 신고(전자민원센터, 소상공인확인서 기반)에 해당하는 것으로 보이나, 이전 세션 원본 대화가 남아있지 않아 100% 확신은 어렵다.
2. "위치정보사업"(더 무거운 쪽, 법인등기부등본+주주명부 필요)은 이번에 처음 명확히 구분해냈다 — 이전 세션에서 다뤘는지 불확실하다.
3. 정부24 상세페이지(gov.kr) 직접 fetch는 인코딩 문제(EUC-KR)로 실패했다 — 검색 스니펫으로 대체 확인했다.
