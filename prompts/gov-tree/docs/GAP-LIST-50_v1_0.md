## B급 — 가설, 미실행 (근거: #11~50 표 구성 중 추론)

### B-1. 레지스트리 대분류 부족 (근거: #22, #30, #42)
`DOCUMENT-TYPE-REGISTRY`가 `IDENT`·`BIZ`·`TAX`·`LICENSE`·`FACILITY`·
`IDENT-FOREIGN`·`PROXY` 7종뿐이다. 건강정보(#22 장애인등록 진단서),
금융정보(#30 부가세 환급 계좌), 통신기록(#42 스팸신고 로그) 모두 기존
대분류에 억지로 끼워 맞춰야 하는 상황으로 추정된다.

### B-2. 순서·선행조건 의존성 미표현 (근거: #25)
INPUT_SCHEMA는 "이 문서가 필요하다"만 표현하고, "이 문서를 받으려면
먼저 다른 승인이 나 있어야 한다"(배출시설 설치허가→폐기물 처리업 등록)
같은 절차 간 순서 관계를 표현할 필드가 없다.

### B-3. 병렬·팬아웃 처리 모델 부재 (근거: #26, #33, #37)
지금까지 GOV_TASK는 전부 "기관 하나와 순차 대화"를 전제한다. 여러 부서
동시 협의(건축허가)나 한 사건을 여러 기관에 동시 통지(폐업신고→세무서
+국민연금+건강보험)하는 구조는 설계에 없다.

### B-4. 장기절차(수개월) 상태관리 모델 부재 (근거: #45, #48)
`gov_tickets`는 짧은 절차를 전제로 설계됐다(G9의 무응답 타임아웃과
구분되는, 정상적으로 수개월 걸리는 절차의 중간 상태 관리). 절차 도중
전제조건이 소멸하는 경우(#48 결혼이민자 국적취득 중 이혼)도 다루지 못한다.

### B-5. 불복·이의제기 절차 완전 공백 (근거: #49) — **가장 중요한 B급 항목**
지금까지 모든 설계(PHASE 0~E, GOV_TASK, HUMAN-AUTHORITY-GATE-SCHEMA)가
"정상 신규 신청" 흐름만 상정한다. 이미 난 행정처분에 불복하는 절차
(세액 부과 이의신청 등, 원처분기관≠심사기관)는 역방향 흐름이라 지금
설계 어디에도 들어갈 자리가 없다.

### B-6. 참여형(비허가형) 절차가 GOV_TASK 범위 밖 (근거: #38, #28)
GOV_TASK는 "신청→발급"류 허가·등록형만 상정한다. 조례 의견수렴처럼
발급물이 없는 참여형 절차(#38), 만료임박 알림처럼 사용자가 요청하지
않아도 기관이 먼저 알려야 하는 절차(#28, 운전면허 갱신)는 범위 밖이다.

### B-7. G8(대리신청)의 과잉차단 가능성 (근거: #16)
G8은 "본인 아닌 자의 대리 요청"을 전부 별도 게이트로 막는데, 법무사·
세무사 같은 **합법적 전문직 대리**는 위임장(`DOC-PROXY-AUTH`)만 있으면
통과해야 정상이다. 지금 G8 문구가 이 정상 케이스와 부정 케이스를
구분하는 기준까지 충분히 명시했는지는 재검토가 필요하다.

### B-8. 응급 판정 경계의 일반화 부족 (근거: #34, #40)
G6(응급 우선)은 "1인칭 당사자의 즉각적 위험"을 기준으로 삼는데, 응급이
끝나고 평시 절차로 전환되는 시점(#34, 재난지원금 신청)이나 대상이
사람이 아닌 경우(#40, 유기동물)의 판정 기준은 아직 없다.

### B-9. 국제/해외 거주자 신원체계 충돌 가능성 (근거: #46)
AUTH_MODEL(3-1, "혼디 로그인=신분증")이 국내 거주를 암묵적으로 전제하는
것으로 보이며, 재외국민의 여권 기반 신원 확인과 어떻게 연결되는지
설계된 바 없다.

### B-10. entity_type=org 전용 tier 판별 규칙 부재 (근거: #44)
JURISDICTION-RESOLVER의 1단계 표는 business/institution 판별 위주로
짜여 있고, 비영리법인(org) 설립허가처럼 entity_type=org에 특화된 판별
규칙은 별도로 검증된 적이 없다.

### B-11. 가족 내 이해상충 처리 모델 부재 (근거: #47)
G8(대리신청)은 "제3자가 타인 행세"를 막는 게이트이지, "가족 구성원
간 이해관계가 서로 충돌하는" 경우(세대분리 시 기존 세대주 동의 필요)를
위한 것이 아니다 — 같은 표면(가족이 관여)이지만 다른 문제로 보인다.

### B-12. 세부 분류 판별 정밀도 이슈 (근거: #18, #20)
"정부산하기관"(준정부기관, #20 근로복지공단)과 "국가경찰/자치경찰"(#18)
처럼, PHASE A·JURISDICTION 1단계의 굵은 범주 안에서 더 세밀한 구분이
필요한 경계 사례들이 있는 것으로 추정된다.
