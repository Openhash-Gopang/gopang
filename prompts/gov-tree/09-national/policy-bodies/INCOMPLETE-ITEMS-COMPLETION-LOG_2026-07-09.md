# 미완료 항목 완성 작업 로그 (2026-07-09)

> 요청: "미완료 항목을 완성하십시오" — 직전 확인된 3개 미완료 항목(정책기관 자동본 34개, 지방정부 SP 22개, 근거법령 22건)에 대한 처리 결과

## 1. 지방정부(제주) SP — 10개 신규 작성 (22개가 아니었음)

착수 전 확인해보니 **22개 중 13개는 이미 `prompts/Jejudo/02-do-dept/`에 `SP-DO-*`로 존재**했다(경제활력국·기획조정실·특별자치행정국 등). 실제 공백은 10개뿐이었다 — 이 확인을 안 했으면 13개를 중복 작성할 뻔했다.

**신규 작성 10개**: 대변인(SPOKES)·소통청렴담당관(COMM)·성평등여성정책관(GENDER)·총무과(GENERAL)·도시균형추진단(BALANCE)·소방안전본부(FIRE)·공항확충지원단(AIRPORTSUP)·강정공동체사업추진단(GANGJEONG)·특별자치제도추진단(AUTONOMY)·중앙협력본부(LIAISON)

기존 `SP-DO-ECON_v1.1.md`의 정확한 문서 구조(§0~§6)를 그대로 복제해 일관성을 유지했다. 특히 두 곳은 **기존 SP와 이름이 헷갈리는 지점을 §1에 명시적으로 경고**해뒀다:
- 소방안전본부(FIRE) ≠ 도민안전건강실(SP-DO-SAFETY, 이미 존재) — 전자는 실제 출동, 후자는 일반 안전정책
- 특별자치제도추진단(AUTONOMY) ≠ 특별자치행정국(SP-DO-JACHI, 이미 존재) — 전자는 제도 확대 추진, 후자는 기존 제도 운영

소방안전본부(FIRE)는 §2에 "응급상황은 절차 설명보다 즉시 119 안내가 최우선"이라는 원칙을 명시 — JEJU-GOV-COMMON의 R0(응급 우선순위) 원칙과 동일한 위계로 다뤘다.

## 2. 정책기관(41개) 자동본 34개 → 전체 보강 완료

기존 7개(ACRC·MOEL·PIPC·FTC·MOHW·MOJ·MOIS)에 이어 **나머지 34개 전부**를 §2(완결 처리 업무)에 도메인 특화 내용을 넣어 보강했다. 이제 **41/41 전부 수작업 보강 완료** — 자동 템플릿 그대로인 게 하나도 없다.

### 보강 과정에서 발견·수정한 문법 버그 (2건)

1. `당신은 **{기관명}**을(를)` — 이번엔 §CAPABILITIES의 "실제 접수·처리 | 안내만 수행 ({채널}로 연결)"에서 채널명 받침 유무에 따라 로/으로가 틀리는 걸 발견. 28개 파일을 자동 스캔해 일괄 수정했다.
2. **수정 중 새 버그를 만들 뻔했다**: 자동 스캔이 "포털"(ㄹ받침 예외 — 받침 있어도 "로"가 맞음)을 놓치고 "포털으로"로 잘못 고쳤다. 재검사로 2건(SUPREMECOURT·NDA) 잡아냈다. **문법 규칙을 코드화할 때도 검증이 필요하다는 걸 다시 확인.**

### 보강 내용의 성격

12개 부처(§ROUTING-PRIORITY-RULES에서 이미 확인한 겹치는 집행기관이 있는 곳)에는 "이 SP와 집행기관 SP는 다른 곳" 경고를 §0에 명시했다. 예: 국방부(MND)는 병무청(SP-NAT-MMA)과, 국토교통부(MOLIT)는 한국공항공사 제주공항(SP-NAT-AIRPORT)과 소관이 겹친다는 걸 각 SP 본문에 직접 적어뒀다 — 라우팅 규칙 문서 따로, SP 본문 따로 다른 말을 하는 걸 방지했다.

## 3. 근거법령 — 22건 그대로 미완료 (의도적으로 중단)

LBOX 조문이동 추적으로 몇 개 더 시도했으나(국토교통부·해양수산부 계열), 검색 결과가 **2023년 이전 개편 전 옛 조번**과 무관한 법령이 뒤섞여 나와 정확도를 담보할 수 없었다. "여성가족부(42조)→국토교통부(43조)→해양수산부(44조)→중소벤처기업부(45조)"라는 옛 순서를 확인했지만, 이게 2025~2026 개편 이후에도 유효한지 확인할 방법을 찾지 못했다.

**여기서 멈춘 이유**: 접수채널과 달리 근거법령은 틀린 조문번호를 기재하면 접수채널 오류보다 훨씬 위험하다(법적 근거를 잘못 인용하는 것). 확신 없이 채우느니 TBD로 정직하게 남기는 게 낫다고 판단했다 — 이건 검색 기반 조사보다 **사람이 law.go.kr 원문을 직접 열어 대조하는 게 맞는 작업**이라는 결론을 재확인했을 뿐이다.
