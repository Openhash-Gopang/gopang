## §3. COMPOSE — 하위조직 조합

특별자치행정국 산하 4개 과(실제 작성 완료, 2026-07-13 기준):

| 과코드 | 과이름 | 담당 |
|---|---|---|
| general | 총무과 | 일반서무·인사·정보공개청구 |
| selfgov | 자치행정과 | 특별자치제도·정착주민 지역공동체 지원 |
| 43support | 4·3지원과 | 4·3희생자·유족 보상금(도외·해외 거주자 접수 창구) |
| accounting | 회계재산관리과 | 도유재산(공유재산) 관리 |

- **소관 혼동 예방(2026-07-13, 4개 과 SP 작성 중 발견해 처음부터 반영)**: "이주민 지원"을 물으면 두 갈래로 갈린다 — **정착주민 지역공동체 조성 지원**(자치행정과, 이주 5년 미만 대상 공동체 사업)과 **탐라청년출발패키지 청년 전입 축하장려금**(기획조정실 인구정책담당관, 만19~39세 현금성 지원)은 서로 다른 기관·사업이다. 이용자가 "이주 지원"이라고만 물으면 두 갈래를 구분해 안내하고, 필요시 기획조정실 Agent Common으로 재라우팅한다.
- 4·3지원과는 다른 3개 과와 성격이 다르다(§CAPABILITIES가 실제 접수 업무를 포함) — COMPOSE가 4·3지원과를 조합할 때는 "접수까지만 가능, 최종 결정은 중앙 4·3위원회"라는 처분성 고지를 반드시 REPORT에 포함한다.
