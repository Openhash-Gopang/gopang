## STEP A — 법정 점검주기·보고기한과 시설 종류별 점검방법 분기

**A-0. 핵심 감별 축 — 절차적 기한과 그 안의 점검방법 분기**
klaw_v15.1(법률 도메인)의 "제1층위(절차적)/제2층위(성격결정)" 명칭 대신, 소방
시설관리 실무의 핵심 감별 축을 쓴다: 법정 점검주기(작동/종합점검)·보고기한처럼
기한이 걸린 절차적 측면(★최우선)과, **시설 종류별(소화/경보/피난/소화활동)
점검방법이 완전히 달라지는** 성격결정적 측면으로 나뉜다.

**[체크리스트 — C26]**
📋 건물 용도·연면적 / 설치 소방시설 현황 / 직전 점검일자·결과 / 보고 대상기관 확인 /
배관 부식·배터리 방전 등 육안 확인 어려운 취약점 점검 이력

- **반대논거(C17)**: "점검표상 '정상'이어도 배터리 방전·배관 부식은 서류만으로는 알 수 없다"

**[위험등급 — C27]**: 🔴[고위험-서류상정상vs실제불량]: 배관부식·소화약제누출·배터리방전 /
🟡[일반점검사항]: 소화기 위치, 표지판 상태

**A-0+. 복수 시나리오 병렬 구성 (C36 적용, v2.4 신설) — 복합용도 건물의 특정소방대상물 해당여부**

소방시설관리사가 반복적으로 마주치는 문제: "1층은 상가, 2~4층은 주택인 건물인데
소방시설을 뭘 갖춰야 하나요?"라는 질문은 **각 용도를 별개로 볼지, 연면적을
합산한 복합건축물로 볼지가 건축물대장·용도별 면적 확인 없이는 확실히 구별되지
않는다.** 이때가 C36이 적용될 지점이다:

- **용도별 개별 판단 시나리오**: 각 용도의 면적이 소규모 기준에 해당하고 별도
  방화구획이 명확한 경우 — 상대적으로 낮은 설치의무가 적용될 수 있음
- **복합건축물 합산 판단 시나리오**: 연면적을 합산하거나 특정 용도(다중이용
  업소 등)가 포함된 경우 — 설치의무가 상향되고 종합점검 대상으로 전환될 수 있음

두 시나리오를 병렬 제시하면서, "건축물대장상 각 층의 용도와 면적이 정확히
어떻게 되는지, 방화구획이 되어 있는지"처럼 **감별에 실제로 쓰이는 확인
사항**을 구체적으로 제시한다(C37, 포괄적으로 "확인이 필요합니다"라고만 하지
않는다).
