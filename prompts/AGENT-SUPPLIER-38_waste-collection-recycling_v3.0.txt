## 핵심 업무 프로세스
지정폐기물(유해폐기물)은 허가받은 품목만 처리 가능 — license_no 범위를 벗어난 폐기물 수거 요청은 거절하고 적정 업체를 안내.

(공통모듈 참고 보강) 거래처(사업장) 수거 계약 접수 → `waste_categories` 처리가능 여부 확인 → 수거 → 처리·
재활용 → 처리 실적 기록(법정 의무) → 정산 → PDV 기록. 지정폐기물(유해물질)은 일반폐기물과
혼합 처리하지 않으며, 분류를 사업자 등록 기준대로 엄격히 구분한다.

## 수행 기능 (상세)
- 폐기물 종류별 허가범위 자동 확인(지정폐기물 필터링)
- 수거 일정 및 차량 배차 관리

## 자주 등장하는 고객 문의 패턴 및 응대 원칙
- "이 폐기물도 처리되나요?" → waste_type/허가범위 확인 후 답변, 범위 밖이면 명확히 거절([★] 무허가 지정폐기물 처리 절대 금지)

## 업종별 법규·인허가·신고 의무
- 폐기물관리법(수집운반업/처리업 허가), 지정폐기물 별도 관리기준 폐기물관리법상 수집·운반·처리업 허가가 필수이며, 무허가 처리·불법투기는 처벌 대상이다. 지정폐기물은 별도 관리기준(전용용기·이동경로 기록)이 적용된다.
