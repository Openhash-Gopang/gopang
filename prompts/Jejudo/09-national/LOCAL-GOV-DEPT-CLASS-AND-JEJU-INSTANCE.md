# 지방정부 부서 클래스 & 제주특별자치도 인스턴스

> 작성일: 2026-07-09
> 목적: `ORG_REGISTRY`(org_profiles, branch=`admin_local`)를 채우기 위한 상위 스키마 정의
> OOP 유비: **클래스** = 대한민국 모든 지방자치단체에 공통되는 부서 구조·근거법령, **인스턴스** = 제주특별자치도가 그 클래스를 실제로 구현한 구체 조직

---

## 1. 클래스 정의 — `LocalGovernmentDept` (추상)

### 1-1. 근거 법령

지방정부 부서는 임의로 이름 붙는 게 아니라 두 법령의 제약을 받습니다.

| 법령 | 규정 내용 |
|---|---|
| 「지방자치법」 제13조 | 지자체가 처리하는 **자치사무의 7대 분야** — 모든 부서명은 결국 이 7개 중 하나로 귀속됨 |
| 「지방자치단체의 행정기구와 정원기준 등에 관한 규정」(대통령령) | 실·국·과 설치 요건(인원·기능 기준), 명칭은 지자체 조례로 자율 결정하되 규모 기준은 이 영을 따름 |

### 1-2. 클래스의 필드 — 지방자치법 제13조 7대 사무 분야

이 7개가 **모든 한국 지자체(광역·기초 불문)에 공통되는 추상 필드**입니다. 실제 부서명(인스턴스 값)은 지자체마다 다르지만, 어느 지자체든 부서를 이 7개 범주 중 하나로 분류할 수 있습니다.

| 필드 ID | 사무 분야 (지방자치법 제13조 원문 기준) | 전형적 부서 계열 |
|---|---|---|
| `F1_ADMIN` | 구역·조직·행정관리에 관한 사무 | 총무과, 기획조정실, 예산담당관, 인사 |
| `F2_WELFARE` | 주민 복지증진에 관한 사무 | 복지가족국, 보건소, 여성가족 |
| `F3_INDUSTRY` | 농림·수산·상공업 등 산업 진흥 | 경제산업국, 농축산식품과, 해양수산과 |
| `F4_DEVELOPMENT` | 지역개발·자연환경보전·생활환경시설 | 건설주택국, 도시계획과, 환경정책과 |
| `F5_CULTURE` | 교육·체육·문화·예술의 진흥 | 문화체육과, 교육지원과 |
| `F6_SAFETY` | 지역민방위·지방소방 | 소방안전본부, 안전정책과 |
| `F7_GLOBAL` | 국제교류 및 협력 | 국제교류과, 통상협력과 |

### 1-3. 클래스의 공통 상위 구조 (모든 지자체 공통 패턴)

부서명은 달라도, 실제 조직 편제는 거의 모든 광역자치단체에서 아래 3계층을 공유합니다.

```
LocalGovernmentDept
├── 보좌기관 (단체장 직속) — 대변인, 감사관, 특별보좌관 등
├── 기획조정 계열 — 총무·기획·예산·인사·법무 (F1)
└── 사업 실·국 계열 — F2~F7 각 분야별 실/국/본부
```

### 1-4. `org_profiles` 스키마 매핑

지난 설계(`worker_orchestration_registry_patch`)의 `org_profiles` 컬렉션 필드에 이 클래스를 얹으면:

```
org_id: {지자체코드}.{부서코드}     예) jeju.economic-vitality
org_branch: admin_local
gov_task_domain: F1~F7 중 택1        (신규 제안 필드 — 사무분야 분류)
jurisdiction: 해당 지자체명
```

## 2. 인스턴스 — 제주특별자치도 (jeju.go.kr 실측, 2026-07 기준)

제주는 「제주특별자치도 설치 및 국제자유도시 조성을 위한 특별법」에 따라 일반 광역자치단체보다 자치권이 넓어(예: 자치경찰, 특별지방행정기관 이양) 부서 구성이 표준보다 크고, 시·군 없이 도가 직접 읍·면·동을 관할하는 단층제라는 점이 다른 광역자치단체와 다릅니다.

### 2-4. 제주 고유 편차 (클래스에서 벗어나는 인스턴스 특이점)

- **단층제**: 시·군 없이 도가 읍·면·동(43개)을 직접 관할 — 기존 43 L1 노드 매핑과 정확히 일치
- **한시조직 다수**: 공항확충지원단·강정공동체사업추진단·특별자치제도추진단은 표준 7대 사무분야로 완전히 환원되지 않는 정책 이슈 대응형 조직 — `org_profiles`에 `is_temporary: true` 같은 별도 플래그가 필요할 수 있음(★ 미구현 ★)
- **자치경찰**: 일반 지자체엔 없는 자치경찰 사무가 있으나 위 실국 목록엔 별도 표기가 없어 확인 필요(★ 검증 필요 ★)
