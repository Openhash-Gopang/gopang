# 혼디 AI 비서 — 신규 200개 사고실험 시나리오 & 대응 루트 상세

> 2026-07-11 작성. AGENT-COMMON §0-A(상식 최우선)·§0-B(정보 해결 3경로: 웹검색·PDV·AI 핸드셰이크)를
> 전제로, 14개 분야(정치·경제·사회·문화·교육·의료·시장·교통·물류·통신·법률·공공·행정·입법)에 걸쳐
> 200개 신규 시나리오와 혼디의 상세 대응 루트를 기술합니다.

## 표기 규약

- **사용자 AI**: 이용자 본인의 그림자 AI(AGENT-COMMON 인스턴스)
- **[GWP: id]**: K 시리즈·정부기관 등 기관 서비스 라우팅 태그
- **[EXPERT: role]**: 전문가 AI(사익 자문) 라우팅 태그
- **핸드셰이크**: §4 절차(신원 고지→권한범위 고지→상대 신원 확인→완료 기록) — 방향(내가 먼저 접촉/상대가 먼저 접촉) 무관하게 동일 적용
- **search_entities**: 위치·업종 기반으로 등록된 기관·업체·개인 AI를 찾는 실제 RPC(현재 K-Market 배송매칭에 실사용 중)
- **PDV**: 이용자 전용 로컬 기록(IndexedDB) — 6하원칙 요약이 GWP 탭 종료마다 자동 축적
- **§0-B 경로1/2/3**: 웹검색 / PDV 조회 / 상대 기관 AI 핸드셰이크

각 시나리오는 "발화 → 대응 루트(단계별) → 도달 SP/서비스 → 비고"로 구성합니다. 비고에는 현재 저장소
실사 결과 알려진 한계(예: 미배선 SP, 데이터 소스 부재)가 있으면 정직하게 표시합니다.

## 정치 (1~14)

### 1. "제주도지사 공약 비교해줘"
- **루트**: 사용자 AI가 §0-A로 정보요청 판단 → §0-B 경로1(웹검색: 후보명·공약 뉴스) → 여러 언론사 보도를 교차 확인해 균형있게 요약
- **도달**: R4 직접 답변(웹검색 보강)

### 2. "이번 지방선거 투표소 어디야"
- **루트**: 사용자 AI가 [현재 위치] 확인 → §0-B 경로1(선관위 홈페이지·검색)로 관할 투표소 조회, 위치정보 없으면 되묻기
- **도달**: R4 직접 답변(웹검색)

### 3. "우리 지역구 국회의원이 누구야"
- **루트**: [현재 위치]로 선거구 특정 → §0-B 경로1 웹검색(국회의원 명단)
- **도달**: R4 직접 답변

### 4. "여야 정책 비교해서 알려줘"
- **루트**: §0-A로 편향 없는 정보요청 판단 → §0-B 경로1 웹검색으로 각 정당 공개 정책자료 수집 → 균형있게 정리(의견 개진 없이 사실 나열)
- **도달**: R4 직접 답변

### 5. "국민동의청원 올리고 싶어"
- **루트**: 사용자 AI가 절차 요청으로 판단 → §0-B 경로1(국회 국민동의청원 홈페이지 절차 검색) → 실제 청원 등록은 본인이 그 시스템에서 직접 수행함을 안내
- **도달**: R4 직접 답변(웹검색) + 실행은 사용자 직접

### 6. "제주도의회 최근 조례 뭐 있어"
- **루트**: 사용자 AI [GWP: jeju] → SP-DO-JACHI(특별자치행정국)로 1차 안내 → 도의회 자체 SP 부재 확인되면 §0-B 경로3(search_entities로 제주도의회 AI 탐색, 있으면 핸드셰이크) 병행, 없으면 경로1 웹검색
- **도달**: [GWP: jeju] → SP-DO-JACHI + 웹검색 보강

### 7. "선거법 위반 같은데 신고하고 싶어"
- **루트**: 사용자 AI가 신고 의도 판단 → [GWP: klaw]로 법률 상담(선거법 조항 확인) → 실제 신고는 선관위 앱/전화 안내
- **도달**: [GWP: klaw]

### 8. "국정감사 일정 알려줘"
- **루트**: §0-B 경로1 웹검색(국회 일정)으로 즉시 답변
- **도달**: R4 직접 답변

### 9. "대통령실에 민원 넣고 싶어"

- **도달**: 정책기관 SP(핸드셰이크 시도) + 웹검색 보강
- **비고**: 정책기관 SP 29개가 최근 신규 작성됐으나 jeju-router.js 렌더링 배선 여부는 별도 확인 필요.

### 12. "정치자금법 최근에 바뀐 거 있어"
