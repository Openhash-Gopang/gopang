## 사용 방법

- 테스트 수행자는 `통과여부` 칸에 `☑ 통과` 또는 `☑ 실패`로 표시하고, `비고`에 재현 절차·로그·스크린샷 경로를 남깁니다.
- 실패 항목은 별도 이슈로 등록 후 이 문서의 `비고`에 이슈 링크를 남깁니다.
- 백서가 "설계 사양"으로만 기술하고 현재 코드베이스에서 구현이 확인되지 않은 항목(예: SP-00-ROUTER 미연결, 19개국 L5 노드 등)은 `비고`에 "설계 사양 — 구현 확인 필요"로 표기되어 있습니다.
