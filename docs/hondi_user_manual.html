일상과 업무에서 혼디에게 "혼디야, ~"라고 말했을 때 실제로 무엇이 처리되는지, 그리고 지금 시점에서 어디까지가 한계인지를 정리했습니다. 상대 기관(국세청·시청·경찰청 등)도 혼디 호환 에이전트를 도입하면 어떻게 달라지는지도 함께 다룹니다.

00 · 이 문서를 읽는 법
가능 범위 · 한계 · 상호운용을 구분해서 봅니다

각 사용 사례는 아래 3가지를 구분해 설명합니다.

현재 가능 범위 지금 혼디가 단독으로 바로 처리할 수 있는 일입니다.
현재 한계 지금은 혼디가 할 수 없거나, 반드시 사용자 확인이 필요한 부분입니다.
🔗 상호운용 시 확장 가능성 — 상대 기관도 혼디(또는 호환 AI 에이전트)를 운영할 경우, 어디까지 자동화되고 어디서부터 여전히 사람이 확인해야 하는지를 보여줍니다.

00-1 · PDV와 AC
아래 모든 사례가 가능한 이유

일상·업무 사례를 보기 전에, 혼디가 이런 일들을 할 수 있는 근본 원리 두 가지를 먼저 알아두면 이해가 훨씬 쉽습니다.

PDV — Private Data Vault 나와 나눈 모든 대화·기록은 서버가 아니라 내 기기(로컬)에 저장됩니다. 서버는 내용을 보관하지 않고, Openhash 기술로 "위변조되지 않았다"는 사실만 증명합니다. 내 데이터는 나 이외에는 누구도 열람할 수 없습니다.
AC — Agent Common 가입 시점에는 모두 같은 AI 비서(AC)로 시작하지만, PDV에 쌓이는 나의 습관·일상·업무 지시를 학습하며 수개월에 걸쳐 개인화됩니다. 시간이 지날수록 문자 그대로 "나만의" 비서가 됩니다.
💡 요약: 이 문서에서 "혼디가 ~을 안다", "혼디가 ~을 기억한다"고 설명하는 부분은 모두 내 PDV 안에서만 일어나는 일입니다. 그리고 혼디가 갈수록 내 말을 더 잘 알아듣는 것처럼 느껴진다면, 그것은 AC가 개인화되고 있다는 증거입니다.

가능 계약서 위험 조항 안내, 보이스피싱 의심 패턴 판단 후 신고 권고(112/금융감독원)
한계 계약이 "불리한지/유리한지"의 최종 법적 판단은 변호사 상담 필요

가능 차량번호·시간·위치·증거 정리해 신고 초안 작성, K-Democracy 익명 투표·의견은 즉시 처리

🔗 환경부·지자체가 혼디를 운영하면 자동 전송·접수확인까지 가능하나, 단속·처분은 담당 공무원의 최종 승인이 원칙입니다.

03 · 공통 원칙
혼디가 지키는 3가지 선

위의 모든 사례에 공통으로 적용되는 원칙입니다.

구분혼디의 역할인간이 반드시 개입하는 지점
정보·추천즉답, 데이터 기반 분석 제공가치 판단(좋다/나쁘다, 합리적이다)은 사용자 본인
의료·법률 자문일반 정보, 절차 안내, 전문가 상담 권유진단, 확정 판결, 수사 처분은 전문가·기관 고유 권한
금전·행정 실행신청서 작성, 신고 초안, 주문 준비발신(사용자 확인)과 승인(기관/담당자 검토) 두 지점은 항상 인간이 직접 처리
💡 요약: 상대 기관이 혼디(또는 호환 AI 에이전트)를 도입하면 "작성 → 전송 → 접수확인"의 속도와 자동화 수준은 크게 높아집니다. 그러나 법적 효력이 발생하는 의사표시(사용자 발신)와 행정처분·진단·판결(기관 측 최종 승인), 이 두 지점만큼은 AI가 아무리 빨라도 인간이 직접 확인하는 것을 원칙으로 합니다.

04 · 가상 시나리오
기관이 혼디를 도입한 미래의 하루

아래는 현재 시점에서는 불가능하지만, 상대 기관(시청·국세청·경찰청·병원 등)이 혼디 호환 에이전트를 도입했다고 가정할 때 어떤 모습이 될지 보여주는 가상 내레이션입니다. 실제 서비스가 아니며, 향후 로드맵 논의를 위한 참고 자료입니다.

① 건축 인허가, 접수까지 9분
"혼디야, 시청에 건축 인허가 문서제출해 줘"라고 말하면 토지대장·설계도면·소유 확인 서류가 자동 정리되어 신청서 초안이 만들어집니다. 제출 확인 후 제주시청 인허가 담당 에이전트로 전송되어 9분 만에 접수 완료 알림이 오고, 3일 뒤 담당 공무원이 현장 여건을 확인해 최종 승인합니다.
✅ 인간이 누른 버튼: ① 사용자의 제출 확인 ② 담당 공무원의 최종 승인
② 사업자등록, 그날 오후에 완료
"혼디야, 국세청에 사업자 등록해줘"라고 말하면 신청서가 작성되고, 본인 서명 후 전송 여부를 확인합니다. 홈택스 에이전트가 자동 검증을 거치고, 세무서 담당자가 소재지·업종 코드를 확인 후 승인하면 그날 오후 사업자등록번호가 발급됩니다.
✅ 인간이 누른 버튼: ① 사용자의 전송 확인 ② 세무서 담당자의 등록 승인
③ 매연 신고, 자동 증거 수집
"혼디야, 앞에 가는 차 매연이 너무 심해. 신고해줘"라고 말하면 차량 번호판·시간·위치·영상이 자동 캡처되어 신고서가 구성됩니다. 사용자 승인 후 환경부 에이전트로 전송되고, 현장 단속 공무원이 실제 검사로 위반 여부를 최종 판정합니다.
✅ 인간이 누른 버튼: ① 사용자의 신고 확인 ② 단속 공무원의 위반 판정
④ 응급실 연계, 진료 전 정보가 먼저 도착
"혼디야, 119"라고 외치면 K-911이 위치를 즉시 전달하고, 사전 동의된 기저질환·복용약물 정보가 응급실 에이전트로 전송됩니다. 구급차 도착 전 의료진이 환자 정보를 미리 확인해 진료를 준비합니다.
✅ 인간이 누른 버튼: ① (사전) 사용자의 의료정보 공유 동의 ② 의료진의 진료 판단
⑤ 보이스피싱 신고와 차단의 자동 연계
의심 통화 신고 시 통화 메타데이터(내용 녹음이 아닌 패턴 정보)가 금융감독원·경찰청 에이전트로 전송됩니다. 경찰청 담당자가 누적 신고 건수를 확인해 수사 착수를 결정하고, 금융감독원은 연계 계좌를 임시 모니터링 대상으로 등록합니다.
✅ 인간이 누른 버튼: ① 사용자의 신고 확인 ② 경찰청 담당자의 수사 착수 판단
⑥ 법인 설립, 정관부터 등기까지
정관 초안이 만들어지면 제휴 법무사 에이전트가 1차 적법성을 검토하고 서명합니다. 이후 등기소 에이전트로 전송되어 등기관이 서류를 최종 심사하면 법인 등록이 완료됩니다.
✅ 인간이 누른 버튼: ① 사용자의 제출 동의 ② 법무사의 1차 검토 서명 ③ 등기관의 최종 등기 승인
⑦ 보험 갱신, 비교 견적부터 가입까지
혼디가 먼저 더 저렴한 보험 견적을 알려주고, 사용자가 동의하면 새 보험사 에이전트로 기존 계약 정보를 전송해 견적을 확정받습니다. 보험사 언더라이터가 운전 경력·사고 이력을 최종 검토해 가입을 승인합니다.
✅ 인간이 누른 버튼: ① 사용자의 가입 확인 ② 보험사 언더라이터의 가입 승인
