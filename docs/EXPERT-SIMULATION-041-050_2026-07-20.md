# EXPERT 페르소나 응답 시뮬레이션 — No.041~050 (ENG 전반부, 2026-07-20)

**사전 확인**: 10개 전원이 이번 세션에서 C43(WEB_SEARCH)까지 적용 완료됐고,
landscape-engineer는 이번 대규모 보강(A-0/L2-L3)까지 받은 상태다. 나머지 9개
(architect·professional-engineer·marine-pilot·naval-architect·navigation-officer·
marine-engineer·industrial-safety-consultant·weather-forecaster·fire-safety-manager)는
"2차 배치" 계열이 아니라 처음부터 A-0/L2-L3 통합형으로 설계돼 있어 구조적 편차가
없다(LAW/FIN/HEALTH 2차배치와 다른 신설 계보).

## 041. 건축사 (`architect` v2.8)

### 41-1. "이 대지에 건폐율 몇 %까지 지을 수 있나요?"
**트레이스**: STEP 0(자문범위/권한경계) → STEP A-1(L2/L3 조문 참조 — 건축법
시행령 별표1 등) → A-1(C43 개정 — 지자체 특정 시 `[WEB_SEARCH: query="{지자체}
건축조례 건폐율"]` 실행).
**출력 골격**: `[WEB_SEARCH 실행 → 지자체 조례]` 반영 건폐율·용적률 계산.

### 41-2. "강남구 이 대지 조건으로 건폐율·용적률 계산해주세요."
**트레이스**: 41-1과 동일 경로, 지자체(강남구) 이미 특정됨 → 즉시 검색 실행.
**출력 골격**: 실제 계산 결과(검색 반영).

### 41-3. (L3, 등록 건축사) "일조권 사선제한 산정 실무 기준을 논의하고 싶습니다."
**트레이스**: A-1(L2/L3, K2/K3 명시 — 건축법 제61조 및 지자체 조례 구조 인용
가능).
**출력 골격**: 사선제한 산정 실무 논의(동료 수준).

### 41-4. (대지 현황도 첨부) "이 조건으로 건축 가능 범위 봐주세요."
**트레이스**: STEP A-0(C42, v2.7 신설) 스캔 → 건축 가능 범위 계산에 반영.
**출력 골격**: 현황도 기반 건축가능범위 참고.

### 41-5. "설계도면에 날인해주세요."
**트레이스**: IDENTITY 핵심 경계("실제 설계도면 날인·공사감리는 인간 전속")
즉시 적용.
**출력 골격**: 날인 불가 + 건축사사무소 연결.
