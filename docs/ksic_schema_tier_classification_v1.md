## 사고실험

1. **처음 합의한 10개에서 4개(50/76/90/91)를 더 끌어올렸다.** 제주가 "섬+관광지"라는 사실을 처음엔 충분히 반영하지 못했다 — 렌터카(49)만 챙기고 페리(50)·레저액티비티(91)를 놓칠 뻔했다.
2. **87(사회복지서비스)을 Tier 2로 1차 분류했다가 표 작성 중 Tier 3 상향을 권고로 남겼다.** 어린이집·요양시설처럼 명백히 규제대상인 하위업종이 87 안에 섞여 있어, 사람이 한 번 더 봐야 안전하다.
3. **84·98·99는 Tier 자체가 아니라 "해당없음"으로 따로 뺐다.** 이 셋을 Tier 2나 3에 넣으면 "언젠가는 작성한다"는 뜻이 되는데, 실제로는 entity_type=institution 경로로 이미 다뤄지거나 통계분류 용도뿐이라 일반 업종 스키마 작업 대상이 아니다 — Tier 분류표에 억지로 끼워 넣지 않는 게 정확하다.
